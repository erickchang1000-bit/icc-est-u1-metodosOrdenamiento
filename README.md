## Practica 1.2 Metodo Burbuja Avanzado
### Fecha: 20/04/2026
### Descripcion:    

Creamos la clase "MetodoBurbujaAvanzado" la cual permite ordenar un arreglo de manera ascendente o descendente de forma más optimizada según se necesite.

La optimizacion consiste en que el algoritmo deja de comparar elementos innecesariamente: cuando detecta que el arreglo ya está ordenado, finaliza el proceso mediante un break, reduciendo así el uso de recursos y mejorando el rendimiento.
